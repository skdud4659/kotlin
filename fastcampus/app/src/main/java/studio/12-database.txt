** sharedPreference
- 키-밸류 방식으로 데이터 저장
- 휴대폰에 데이터를 저장
- 데이터는 앱 밑에 있다.
    > 앱을 삭제하게되면 데이터도 삭제.

- 사용자가 앱을 이용하는데에 있어 간단한 기호 데이터 등을 저장해야한다.
    - 폰트 크기 등
- 앱끼리 공유할 수 있음(ex. 카카오페이, 카카오뱅크, 카카오톡 등 연관된 앱끼리)

** room
- 데이터베이스를 쉽게 사용하기 위한 API
- room을 이용해서 SQLite을 조작
- 안드로이드 데이터 저장방식 (내장)
    - sharedPreference(키-밸류)
    - SQLite(관계형)
- 장점
    - 데이터베이스(관계형 데이터 베이스)를 앱(관계형 데이터 베이스)에 캐싱
        - 서버의 부하를 줄일 수 있음
        - 오프라인에서 남은 데이터 확인이 가능하여 사용할 수 있음
        - 속도를 줄일 수 있음
