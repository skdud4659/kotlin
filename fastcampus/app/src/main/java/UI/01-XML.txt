** General Purpose Language
- 범용적인 목적을 가지고 있는 언어
- kotlin, python, java, c, c++, swift

** Domain Specific Language (DSL)
- 도메인 특화 언어 > 특수한 목적을 달성하기 위한 언어
- 간단한 문법
- xml

** xml(Extensible Markup Language)
- 안드로이드에서 UI를 그리기 위해 채택한 언어
- Markup Language
    - 태그로 마크를 하여 내용을 작성
    - 태그 > 범위
    - Extansible > 태그 안에 태그 추가 가능
