- drawable
    - image(실제 이미지 파일, xml)
    - 동일한 이미지를 해상도별로 여러장을 준비할 수 있다.
        - 핸드폰의 해상도에 맞는 이미지 사용
        - 안드로이드OS가 적합한 이미지를 알아서 선택
        - 같은 폴더 내에 위치해야 함.
    - 이미지 리소스
        - .jpg, .png : 복잡한 이미지
        - xml : 간단한 이미지, 도형
